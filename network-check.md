# Case: Network Connectivity Issue

## Symptoms

- Users cannot access Tableau Server
- Intermittent connection issue
- Slow response from server

## Initial Checks

- Ping server
- Check DNS resolution
- Validate port accessibility
- Check firewall or network changes
- Confirm whether issue is local or widespread

## Troubleshooting Steps

1. Validate connectivity from user side and server side.
2. Check DNS and IP resolution.
3. Test relevant ports.
4. Review recent network or firewall changes.
5. Escalate to network team if needed.

## Prevention

- Document network dependencies.
- Maintain escalation contact.
- Include connectivity checks in maintenance activity.
