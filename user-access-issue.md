# Case: User Access Issue

## Symptoms

- User cannot log in
- User cannot access specific content
- Permission error appears

## Initial Checks

- Validate user account status
- Check user role
- Check group membership
- Check project/workbook permissions
- Confirm whether issue affects one user or multiple users

## Troubleshooting Steps

1. Confirm user identity and affected content.
2. Review permissions.
3. Check group membership.
4. Compare access with another user if needed.
5. Apply permission correction if approved.
6. Ask user to retest access.

## Resolution

Document permission changes and confirmation from user.

## Prevention

- Review permission standards.
- Maintain access request documentation.
