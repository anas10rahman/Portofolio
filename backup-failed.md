# Case: Tableau Server Backup Failed

## Symptoms

- Backup process fails
- Backup file is not generated
- Error appears during maintenance window

## Initial Checks

- Check available disk space
- Validate backup destination
- Review TSM backup command output
- Check Tableau Server status
- Review backup-related logs

## Troubleshooting Steps

1. Confirm the backup command and parameters.
2. Validate disk capacity.
3. Confirm access to backup destination.
4. Review error messages.
5. Retry backup after resolving the identified issue.
6. Escalate if failure persists.

## Resolution

Document root cause, action taken, and backup validation result.

## Prevention

- Monitor backup storage.
- Validate backup schedule.
- Review backup retention policy.
