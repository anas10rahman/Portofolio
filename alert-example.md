# Alert Example

## Condition

Disk usage exceeds 80%.

## Impact

High disk usage may cause Tableau Server performance issues, failed extract refreshes, failed backups, or service instability.

## Recommended Action

1. Identify large files or log directories.
2. Archive or clean old logs according to policy.
3. Validate available disk space.
4. Monitor disk growth after cleanup.
5. Document the action in the maintenance report.
