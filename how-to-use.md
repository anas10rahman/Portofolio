# How to Use

1. Run the scripts on a Linux environment.
2. Validate disk, CPU, memory, and Tableau Server service status.
3. Generate a report using `generate_health_report.sh`.
4. Review the generated report and add findings or recommendations.
5. Attach the final report to preventive maintenance documentation.

## Important Notes

- Mask or remove confidential information before sharing any report.
- Do not upload real production logs to public repositories.
