# Tableau Server Preventive Maintenance Checklist

## 1. Server Resource Check

- [ ] Check CPU usage
- [ ] Check memory usage
- [ ] Check disk capacity
- [ ] Check server uptime
- [ ] Check network connectivity

## 2. Tableau Services Check

- [ ] Verify Tableau Server overall status
- [ ] Check gateway status
- [ ] Check repository status
- [ ] Check backgrounder status
- [ ] Check vizqlserver status
- [ ] Check dataserver status

## 3. Backup and Recovery Check

- [ ] Verify latest backup availability
- [ ] Confirm backup file location
- [ ] Validate backup schedule
- [ ] Confirm backup storage capacity

## 4. Log and Capacity Check

- [ ] Review log directory size
- [ ] Identify abnormal log growth
- [ ] Archive or clean old logs according to policy

## 5. Documentation

- [ ] Record findings
- [ ] List identified risks
- [ ] Provide recommendations
- [ ] Submit maintenance report
